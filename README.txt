Zeldiablo

Projet universitaire réalisé par :
    - Bernard Hugo
    - Rethiers Aurélien
    - Gachenot Antoine
    - Morel Erwin
    - Gregoire Dylan


Il s'agit d'un petit jeu codé en Java qui peut s'apparenter dans le type de jeu à un ancien zelda.

On peut se déplacer, donner des coups.
Il existe plusieures salles (ou cartes) avec des transitions pour passer de l'une à l'autre.
Certaines salles sont vides, d'autres sont remplies de monstres de différentes sortes et une salle renferme le boss final.
Le but du jeu est de trouver la salle du boss en se déplaçant de salle en salle pour aller le tuer et gagner la partie.
Toutefois si vous tombez à 0 points de vie, vous mourrez.


Vous trouverez le code du jeu dans le dossier projet_zeldiablo et les diagrammes uml du projet dans documents.
